1. Создать файл “new.json” -> (touch new.json)
2. Добавить файл под гит. -> (git add new.json)
3. Закоммитить файл. -> (git commit -m "add new.json")
4. Отправить файл на внешний GitHub репозиторий. -> (git push)
5. Создать файлы new2.json, new2.xml, new2.txt -> (touch new2.json new2.xml new2.txt)
6. Добавить файлы под гит. -> (git add new2.json new2.xml new2.txt)
7. Закоммитить файлы. -> (git commit -m "add new2.json, new2.xml, new2.txt")
8. Отправить файлы на внешний GitHub репозиторий. -> (git push)
9. Отредактировать содержание файла “new2.txt” - написать информацию о себе (ФИО, возраст, количество домашних животных, будущая желаемая зарплата).

10. Отредактировать содержание файла “new2.json” - написать информацию о себе (ФИО, возраст, количество домашних животных, будущая желаемая зарплата). Всё написать в формате JSON.

cat > ./new2.json <<EOF
{ 
	"Name": "Mitskevich Dmitry",
	"Age": 39,
	"Pets": 0,
	"Guba": "6000$ - да да, губа не треснет :)"
}
EOF

11. Добавить и закоммитить “new2.txt” одной строчкой. -> (git commit -am "change new2.json, new2.txt") или (git add new2.txt&&git commit -m "change new2.txt")
12. Добавить и закоммитить “new2.json” одной строчкой.-> выполнил одним запросом сразу 2 файла (см.выше) или (git add new2.json&&git commit -m "change new2.json")
13. Отправить изменения на внешний репозиторий. -> (git push)
14. Отредактировать содержание файла “new2.xml” - написать информацию о себе (ФИО, возраст, количество домашних животных, будущая желаемая зарплата). Всё написать в формате XML.

<item><Pets/>0</item>
<item><Guba/>6000$ - да да, губа не треснет :)</item>
